Femboy cafe

I'm developing a comprehensive Tycoon-style simulation game on the Roblox platform using Luau scripting. The project focuses on creating an engaging gameplay loop that balances player progression with overall economy management. The core concept centers on an “Emotional Loop”—Aspiration, Build, Reward, and Progress—to support strong player retention.

EXE:ETERNAL

This project belongs to the 2D Action-Platformer & Soulslike genre. Set within a narrative Time Loop, the core gameplay focuses on High-Precision Combat mechanics. Players must master Stamina Management and Frame-Perfect Parrying to survive. The difficulty curve relies on Pattern Recognition, requiring players to analyze and adapt to complex enemy AI behaviors in a strictly Single-Player environment.

Visual Hygiene Quantification System

I developed a Python script that quantifies image characteristics using the OpenCV library. The algorithm applies Otsu’s thresholding to automatically separate the foreground from the background based on pixel intensity. It then computes the pixel-density ratio to generate a “Cleanliness Percentage,” enabling automated visual assessment using computer vision techniques.

Automated Plant Watering System using Soil Moisture Sensors

The system is built on the Arduino R3 microcontroller platform and uses a soil-moisture sensor to continuously measure water content in the soil. I program the device in C/C++ and developed an algorithm that automatically activates the water-pump relay when the moisture level drops below a defined threshold.
